# Front-End Notes

A collection of front-end development notes for HTML, CSS, responsive design, and styling methodologies.

## Contents

- `html.md` — HTML structure, semantic tags, boilerplate, and file paths
- `css.md` — CSS positioning, floats, Flexbox, Grid, and alignment
- `responsive-design.md` — media queries, mobile-first patterns, and responsive layout strategies
- `css-methodologies.md` — why CSS methodologies matter and how they improve maintainability

- Concepts and best practices
- Framework and library summaries
- HTML, CSS, and JavaScript patterns
- UI design notes and prototyping tips



## Usage

Use this repository to organize front-end reference notes in separate topic files.

- Keep `README.md` as the top-level index.
- Add detailed notes in the separate markdown files.
- Update the topic files as you learn new front-end patterns and techniques.

## Suggested workflow

1. Read the overview in `README.md`.
2. Open `html.md` or `css.md` for focused reference material.
3. Use `responsive-design.md` for layout and breakpoint guidance.
4. Refer to `css-methodologies.md` when organizing larger stylesheets.
